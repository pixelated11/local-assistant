# LocalLLM
Run Llama 3 8B large language model locally, with your hardware, with your control. Write AI prompt, and see the response _privately_. Completely offline.
> [!Warning]
> This repository is not ready yet. It is under development. I published it publicly because I need to work with other people, using git.
## Build, manual, and infos
You can see the build instructions, manual on how to use it, and informations on the documentation page [here](https://pixelated11.github.io/localllm-docs)
